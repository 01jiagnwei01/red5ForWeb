red51.5需要jdk使用1.7+
windows 安装方法：
1：下载
	1.1从网址下载window安装文件。https://github.com/Red5/red5-server
2：安装
	2.1 双击下载的文件，next by next
	2.2 设置red5安装目录：<red5_home>=c:/red5
	2.3 设置IP：127.0.0.1
	2.4 设置端口：80
3:启动
	3.1 到<red5_home>,双击red5.bat进行启动。如果出现闪退，则到red5.bat文件，在最后一行后追加一行，写入：pause
	3.2 到<red5_home>,双击red5-debug.bat进行启动。
